

# NeuroCardiaSync – Behavioral Signal Correlation Analysis

This is a simple Python-based data analysis project developed to understand and analyze the relationship between behavioral patterns and cardiac signals.
The project focuses on correlating brain–heart related data using basic machine learning techniques.

## Features

* Behavioral and cardiac data analysis
* Signal correlation and pattern detection
* Machine learning models (MLP, Bagging)
* Simple and easy-to-follow notebook flow

## Technologies Used

* Python
* Jupyter Notebook
* Pandas, NumPy
* Scikit-learn
* Basic Machine Learning concepts

## Purpose

This project is created for learning and academic purposes. It helps beginners understand how behavioral and physiological data can be analyzed and correlated using Python and machine learning techniques.

## How to Run

* Clone the repository
* Open the project in Jupyter Notebook or any Python IDE
* Run the notebooks step by step
